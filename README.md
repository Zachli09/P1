# Store Management System

A console-based C++ application designed to help a store manage daily sales transactions and customer payments.

## Features

* User login system with limited login attempts
* Track daily sales totals
* Add sales with automatic tax calculation
* Process customer refunds
* Calculate cash change using the least number of bills
* Menu-driven interface for user interaction

## Technologies Used

* C++
* Functions
* Loops and Conditionals
* Input Validation
* Console-Based User Interface

This project helped me improve my understanding of program structure, user authentication systems, menu-based applications, and problem-solving using C++.
